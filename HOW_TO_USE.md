## üé® HOW TO CUSTOMIZE IT

### Change the Color

**Step 1**: Open `styles.css` with any text editor

**Step 2**: Find this line:
```css
background-color: #3B82F6;
```

**Step 3**: Replace `#3B82F6` with your color:
```css
background-color: #FF6B6B;  /* Your new color */
```

**Step 4**: Save the file

**Step 5**: Refresh your browser (press F5)

**Colors to try**:
- `#E74C3C` - Red
- `#27AE60` - Green
- `#2980B9` - Blue
- `#8E44AD` - Purple
- `#F39C12` - Orange

### Change the Text

**Step 1**: Open `index.html` with any text editor

**Step 2**: Find the text you want to change

**Step 3**: Edit it:
```html
<h1>Old Title</h1>
<!-- becomes -->
<h1>New Title</h1>
```

**Step 4**: Save the file

**Step 5**: Refresh your browser

### Change the Logo

**Step 1**: Open `index.html`

**Step 2**: Find:
```html
<span class="logo-text">Corporate Modernism</span>
```

**Step 3**: Change to:
```html
<span class="logo-text">Your Company</span>
```

**Step 4**: Save and refresh
